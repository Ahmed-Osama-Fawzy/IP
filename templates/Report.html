Project Definition

This report outlines an image processing project aimed at detecting text regions within images using a combination of preprocessing techniques, segmentation, clustering, and bounding box merging. The project leverages Python libraries such as OpenCV, NumPy, and scikit-learn to preprocess images, detect text regions using the Maximally Stable Extremal Regions (MSER) algorithm, cluster these regions using DBSCAN, and merge overlapping bounding boxes to accurately identify text areas. The methodology is demonstrated on sample images, and the results are visualized to evaluate the effectiveness of the approach.



Problem Definition

Text detection in images is a fundamental task in computer vision with applications in optical character recognition (OCR), document analysis, and automated data extraction. However, variations in lighting, noise, and complex backgrounds make accurate text detection challenging. This project addresses the problem of identifying and localizing text regions in images with varying conditions

Objectives

Preprocessing

Enhance image quality to improve text detection accuracy using techniques such as histogram equalization and morphological operations.

Text Region Detection

Use the MSER algorithm to identify potential text regions.

Clustering and Merging

Apply DBSCAN to cluster detected regions and merge overlapping bounding boxes for better accuracy.

Visualization

Display the original and processed images with detected text regions for evaluation.

Text Region Detection (MSER)

is used to detect text regions by identifying stable connected components in the image, The algorithm returns bounding boxes around potential text regions

Clustering (DBSCAN)

groups nearby bounding boxes into clusters based on spatial proximity, Helps eliminate false positives and merge overlapping detections

Bounding Box Merging

  • For each cluster, compute the minimum and maximum coordinates to form a single merged bounding box.
  • Ensures that fragmented text regions are combined into a single detection.

Visualization

  • Draw the merged bounding boxes on the original image.
  • Compare the original and processed images to evaluate detection accuracy.
